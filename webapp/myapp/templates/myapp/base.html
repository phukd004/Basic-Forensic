Requirement to Use This Program

Tool and Framework

Volatility 3 Framework Link

after extracting file to "tools" folder, this is way to enable the full range of volatility3 functionality in folder named "volatility3"

pip3 install -r requirements.txt

CheatSheet (if you want to use others command) :

Basically, how to use plugins of this framework is :

python vol.py -f "path/to/file" [OS].[plugin].[Plugin]

if you want to know plugin option :

python vol.py [OS].[plugin].[Plugin] -h

Set up & Tutorial

By defualt, PowerShell execution policy is set to Restricted; that means script can't run on this computer.

Command to verify the execution policy setting by using this command

Get-ExecutionPolicy

Then the command to change the PowerShell script execution policy for using. (run as Administrator)

Set-ExecutionPolicy RemoteSigned  *after use the script, please turn execution policy to Restricted*

Now, you can run script on terminal. (run as Administrator)

.\insightify.ps1
